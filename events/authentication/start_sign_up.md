# Start Sign Up

Fire whenever a user views the first step of the account creation form.

## Javascript Code

```js
window.dataLayer = window.dataLayer || [];
dataLayer.push({ event_data: null });  // Clear the previous event_data object.
dataLayer.push({
  event: 'start_sign_up'
});
```
